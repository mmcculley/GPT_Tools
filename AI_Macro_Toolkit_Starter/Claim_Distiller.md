# 🔍 Claim Distiller

## 🔹 Purpose
Extract clear, testable claims from verbose or symbolic input. Useful before SR, TPM, or logical analysis.

---

## ✅ How to Use

### 🔁 Trigger:
```
Distill Claim: [insert text]
```

### 📤 Output:
- 1–3 explicit epistemic claims

---

## 🧩 Overlay Compatibility
- ✅ SR Mode
- ✅ TPM
